Instalar Windows nunca ha sido tan fácil

La instalación de Windows 8 ha cambiado comparada con versiones anteriores del sistema operativo. Instalar Windows nunca había sido tan fácil. Sigue estos pasos para lograrlo.

El proceso de instalación de Windows 8 ha cambiado radicalmente con respecto a Windows 7. Mientras que la parte de instalación en el disco duro es relativamente similar, con nuestro primer arranque entraremos en un proceso de configuración novedoso en Windows y que nos permitirá adaptar el sistema a nuestras necesidades.

Paso 1 – Selección de idioma y cultura

Una vez hayamos arrancado desde el DVD o USB que contenga Windows 8, la primera pantalla que veremos nos permite seleccionar los siguientes valores:

  • Idioma de la instalación [1]: determinará el idioma de menús, mensajes de error, ayuda, etc.
  • Cultura [2]: afecta a aspectos como el formato de fecha y números, moneda, etc.
  • Idioma del teclado [3]: nos permite que las teclas pulsadas se reflejen como se espera. Si nuestro teclado no es español deberemos cambiar esta opción.

Tanto en este paso como en la mayoría de los que veremos hasta completar el proceso, para avanzar a la siguiente pantalla deberemos pulsar Siguiente.

Pasos 2 – Instalación, Clave y Aceptación de Contrato

Los siguientes pasos son muy sencillos. Primero seleccionamos Instalar ahora para hacer una instalación desde cero, en lugar de una reparación.

 

Posteriormente introducimos la clave de producto que habremos recibido junto con la versión de Windows adquirida. Y, por último, nos leemos el contrato de Windows (sin duda la parte más dura de toda la instalación…) y lo aceptamos para poder continuar con el proceso.

Paso 3 – Tipo de instalación

Existen dos tipos de instalaciones en Windows desde hace bastantes versiones: actualización [1] e instalación desde cero [2].

La actualización es posible en ciertos escenarios y nos permite, en teoría, no perder nada de lo que tuviéramos instalado anteriormente, reaprovechándose aplicaciones, configuraciones, personalizaciones, etc.

Frente a esta opción está la instalación desde cero, más costosa en tiempo puesto que nos obliga a instalar y configurar de nuevo muchas opciones. Sin embargo, en mi experiencia la estabilidad con instalaciones desde cero es siempre mejor, de ahí que prefiera invertir un poco más de tiempo en cada instalación de Windows (algo que tampoco se hace muy frecuentemente) a cambio de tener la mejor experiencia de uso posible a posteriori.

Paso 4 – Dónde instalar Windows

En este paso seleccionaremos dónde vamos a instalar Windows. Si, como en este caso, sólo tenemos un disco duro [1] y además está sin particionar, simplemente tendremos que seleccionarlo y pulsar Siguiente.

Por otro lado, si ya tenemos particiones o queremos crearlas/eliminarlas, podremos utilizar las opciones avanzadas Nuevo, Eliminar y Formatear [2] para proceder a su manipulación y adecuación a nuestras necesidades.

Una vez que pulsemos Siguiente se iniciará el proceso de instalación, que no requiere de la intervención del usuario y que se completa en pocos minutos. Al terminar, la máquina se reiniciará y arrancará en el sistema operativo recién instalado.

Paso 5 – Elige tu color y tu nombre de máquina

A partir de este paso el proceso de instalación pasa a convertirse en una personalización para adecuar el sistema operativo a nuestros gustos y preferencias.

En este primer paso podremos elegir el color general para el sistema [1] y el nombre de la máquina [2]. Con el deslizador podremos cambiar el color y ver su efecto aplicado a la propia ventana, para hacernos una idea.


Paso 6 – Tipo de configuración

El proceso de configuración puede realizarse en dos modos: configuración rápida [1] o personalizada [2].

Si elegimos configuración personalizada tendremos los siguientes pasos extra:

  • Configurar el tipo de red al que estamos conectados, que afecta a la forma en que se comparten (o no) recursos compartidos como carpetas o impresoras.
  • Decidir si Windows Update debe instalar automáticamente las actualizaciones disponibles y si confiamos en SmartScreen para vigilar el tipo de aplicaciones que bajamos y las páginas a las que navegamos, para prevenirnos de virus, phising, troyanos y otras amenazas de la red para nuestra seguridad.
  • Definir qué información queremos compartir con Microsoft, de forma anónima, cuyo objetivo es ayudar a mejorar la experiencia de uso de Windows 8.
  • Decidir a qué información personal pueden acceder las aplicaciones, como ubicación o nombre e imagen.

Personalmente recomendaría siempre la configuración rápida puesto que los valores que configura son los más recomendables para cada uno de los pasos por los que pasaremos si elegimos la configuración personalizada.

Paso 7 – Cuenta Windows

En Windows podremos utilizar dos tipos de cuentas para iniciar sesión:

  • Cuentas locales [1]: son el tipo de cuentas que existían hasta esta versión. Se crean localmente y no pueden utilizarse para iniciar sesión en otras máquinas, como su propio nombre sugiere.
  • Cuentas Microsoft [2]: están asociadas a un correo de Microsoft, como Hotmail, MSN u Outlook. Ofrecen una experiencia de uso superior a las cuentas locales porque podemos utilizarlas en distintas máquinas, entre las que sincronizarán nuestras preferencias, configuraciones, aplicaciones instaladas, etc. Además son necesarias para poder bajar aplicaciones de la Tienda.

Como es lógico la opción recomendada es la cuenta Microsoft. Una vez que hayamos introducido la dirección de mail se nos pedirá la contraseña para verificar que somos sus legítimos dueños. Este paso hará que la nueva máquina se asocie con nuestra cuenta. Recibiremos, por ello, un mail para confirmar que confiamos en dicha máquina.

Paso 8 – Completando la configuración

Tras autenticarnos satisfactoriamente, Windows 8 realizará unos últimos ajustes automáticamente mientras nos muestra un breve tutorial de uso. Terminado dicho paso tendremos acceso por primera vez a nuestra pantalla Inicio.

En ella podremos ver las aplicaciones por defecto de Windows 8 y, en función de si hemos conectado previamente nuestra cuenta Microsoft con redes sociales como Facebook, también nos aparecerá información de nuestros contactos.
