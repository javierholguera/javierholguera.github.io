Utiliza Windows 8 como ya estás acostumbrado a hacerlo

Aunque Microsoft ha emprendido toda una revolución con Windows 8, sigue habiendo espacio para hacer las cosas como estábamos acostumbrados en versiones anteriores.

Si eres usuario de versiones anteriores de Windows y no eres amigo de los cambios puede que Windows 8 te intimide; o quizás utilizas Windows en un ambiente profesional y no te puedes permitir semanas de experimentación y aprendizaje. Afortunadamente Microsoft ha pensado en esos usuarios que tenemos nuestra vida profesional y personal organizada en torno a nuestros ordenadores. Para ellos ha añadido una aplicación en Windows 8 que nos da acceso, como si de una máquina del tiempo se tratara, a una versión de Windows muy similar a lo que teníamos en Windows 7, incluida total compatibilidad con sus aplicaciones. Esta aplicación se llama Escritorio.

1. El Escritorio es una aplicación Metro

Lo primero que tenemos que tener claro es el Escritorio es una aplicación Metro como cualquier otra. Se ejecuta a pantalla completa, se integra con los charms, implementa una vista acoplada, aparece como cualquier aplicación Metro en la Barra de Aplicaciones, tiene su propio Tile y se puede lanzar desde la Pantalla de Inicio, se puede cerrar con los mismos mecanismos que ya hemos visto, etc.

Esta aplicación Metro imita con gran exactitud el funcionamiento de Windows 7: permite ejecutar cualquier aplicación de escritorio, tiene una barra de tareas, un panel de control, etc. En definitiva, se comporta como si tratara de una máquina virtual ejecutando Windows 7, pero perfectamente integrada con el resto del sistema.

A pesar de todo, existen algunas diferencias con respecto a Windows 7, algunas de ellas lo suficientemente relevantes como para que su uso no sea exactamente equivalente.

2. Adiós al Menú de Inicio

Seguramente sea la decisión más polémica que ha tomado Microsoft en Windows 8 y afecta directamente al Escritorio: no existe Menú de Inicio. Aunque incluye una barra de tareas idéntica a la de Windows 7, al principio de la misma no existe el clásico botón de Windows que daba acceso a programas, documentos, enlaces más frecuentes u opciones de apagado.

Este menú ha desaparecido, según Microsoft porque ya no es necesario. Tenemos una opción mejor en la Pantalla de Inicio, desde la que podemos lanzar aplicaciones, como hacíamos en Windows 7, y además recibir información dinámica a través de los tiles.

3. Menú de Inicio simplificado

Si no eres capaz de sobreponerte al shock de la desaparición del Menú de Inicio o simplemente no te adaptas a la nueva filosofía, por suerte existen alternativas. La primera, la más fácil pero menos completa, está presente en el propio sistema operativo si lo manejas con teclado y ratón.

Pulsando las teclas WIN y X o llevando el ratón a la esquina inferior izquierda y pulsando el botón derecho, aparecerá una versión simplificada del Menú de Inicio.

En esta versión encontraremos enlaces a las opciones más importantes del sistema, como el Panel de Control, la información del Sistema, Programas y Características, etc. En definitiva, si te has acostumbrado a usar la barra de tareas para lanzar tus aplicaciones y el Menú de Inicio sólo lo utilizabas para la gestión del propio sistema operativo, con este menú tendrás tus necesidades cubiertas.

4. Utilizar una aplicación de terceros

La otra opción que existe para recuperar el Menú de Inicio es utilizar alguna de las aplicaciones de terceros que se han lanzado al mercado desde la aparición de Windows 8 y la consiguiente avalancha de quejas por la ausencia del mismo. La oferta es variada y va desde aplicaciones gratuitas que “cumplen” a aplicaciones de pago realmente logradas.

Una de las mejores opciones, por resultados y por su gratuidad, es Classic Shell. Una vez instalada veremos, justo a la izquierda de la barra de tareas (donde solía encontrarse el Menú de Inicio), un nuevo botón. La primera vez que pulsemos en él nos dará la opción de elegir entre varios estilos de menú. Si seleccionamos Windows 7, a partir de ese momento contaremos con un menú muy similar al que conocemos.

En cualquier caso, Microsoft ha prometido recuperar, en la próxima actualización del sistema operativo, el botón del Menú de Inicio. Hablaremos más en profundidad de ello en una sección posterior dedicada a las novedades de Windows 8.1.

5. La estética también ha cambiado

Los cambios estéticos entre el Escritorio de Windows 8 y Windows 7 no son muy marcados, pero existen. Aero, el tema gráfico utilizado en Windows 7, se ha metroficado en Windows 8; ha simplificado sus líneas, eliminado transparencias, utilizado colores más sólidos, etc. En general, resulta más armonioso con este lenguaje de diseño que ha regido el desarrollo de Windows 8.

Otro de los cambios más relevantes en el aspecto gráfico es la introducción de la barra de menús Ribbon en el Explorador de Ficheros. Este tipo de barra, que hizo su debut con Office 2007, se caracteriza por utilizar pestañas para organizar los distintos menús, por el uso de iconos y por generar controversia cada vez que una aplicación la adopta.

Si no nos acostumbrados a Ribbon en el Explorador de Ficheros podemos utilizar una aplicación gratuita llamada Ribbon Disabler for Windows 8 para desactivarlo.
